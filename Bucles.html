#Deber 2 - Bucles
#Imprime los números del 1 al 10
for i in range(1, 11):
    print(i)

#Imprime los números pares del 1 al 20
for i in range(1, 21):
    if i % 2 == 0:
        print(i)

#Muestra la tabla de multiplicar del número que ingrese el usuario
num = int(input("Ingrese un número: "))

for i in range(1, 11):
    print(num, "x", i, "=", num * i)

#Imprime los números del 10 al 1 en orden descendente.
for i in range(10, 0, -1):
    print(i)

#Pide un número y calcula la suma acumulada desde 1 hasta ese valor.
n = int(input("Ingrese un número: "))
suma = 0
for i in range(1, n + 1):
    suma += 1
    print("La suma es: ", suma)

#Cuenta cuántas vocales hay en una palabra ingresada por el usuario.
palabra = input("Ingresa una palabra: ")

contador = 0
for letra in palabra:
    if letra in "aeiou""AEIOU":
        contador += 1 
print("Cantidad de vocales: ", contador)   

#Solicita una contraseña repetidamente hasta que el usuario la ingrese de forma correcta
correcta = "Alex1234"

while True:
    clave = input("Ingrese la clave: ")
    
    if clave == correcta:
        print("Correcto")
        break 
    else:
        print("Incorrecto, intente otra vez")

#Muestra los números del 1 al 50 que sean múltiplos de 5
for i in range(1, 51):
    if i % 5 == 0:
        print(i)

#Pide 5 números al usuario e indica cuál es el mayor.
mayor = None
for i in range(5):
    num = float(input("Ingresa el número: "))
    
    if mayor is None or num > mayor:
        mayor = num
print("El número mayor es: ", mayor)

#Imprime un triángulo de asteriscos de 5 filas (* / ** / *** …)
for i in range(1, 6):
    print("*" * i)

El deber 1 trata sobre la Bucles for y while, utilizada en programación Repetir varias veces un algo de acuerdo a un rango o hasta que una condición se cumpla.

El Bucle "for" se usa cuando sabes cuántas veces repetir algo. El bucle "while" se usa cuando no sabemos cuantas veces repetir.
